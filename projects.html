Here are some of the projects I did for classes at Stanford University.

Sparse + Low-rank Approximation of Matrix Inverse

EE 364B: Convex Optimization II Class Project with Nipun Agarwala and Shaked Regev

[paper] [code]

We proposed and implemented a scalable convex optimization approach to estimate a low-rank correction of a sparse approximate matrix inverse. Our approach can improve the preconditioning matrix in the Preconditioned Conjugate Gradient algorithm.

Training CNN for Mixed Poisson-Gaussian Noisy Images Without Ground Truth

CS 229: Machine Learning Class Project with Patin Inkaew

[paper] [code]

We used an unbiased risk estimate for mixed Poisson-Gaussian noise as training loss instead of the squared error loss. This approach allows for training of CNN based denoisers without requiring ground truth images when the noise is mixed Poisson-Gaussian. This approach enables finetuning on datasets in which ground truth data are not available.

Variants of SURE-LET Approach to Image Denoising

EE 367: Computational Imaging and Display Class Project

[paper]

I experimented with some instances of the SURE-LET approach for blind Gaussian denoising and PURE-LET, the variant for Poisson noise case. In these approaches, we apply a denoiser to an image using different noise strength parameters and linearly combine the denoised images such that the unbiased estimate of the mean squared error (MSE) is minimized. I also experimented with using Anscombe transform to approximate a Poisson noisy image with a Gaussian noisy image and denoise it using the SURE-LET approach.
